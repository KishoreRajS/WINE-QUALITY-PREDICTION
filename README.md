# Wine Quality Prediction using Machine Learning

## 📌 Project Overview
This project focuses on predicting the **quality of wine** based on its chemical characteristics.  
It demonstrates a real-world application of **machine learning classification techniques** in the context of viticulture.

Multiple classifier models are implemented and compared to evaluate their effectiveness in predicting wine quality.

This project is completed as **Project – Level 2** of the **Oasis Infobyte Data Analytics Internship**.

---

## 📂 Dataset Description
The dataset consists of various **chemical attributes of wine**, which are used as features to predict wine quality.

Key features include:
- Fixed Acidity
- Volatile Acidity
- Citric Acid
- Residual Sugar
- Chlorides
- Density
- pH
- Sulphates
- Alcohol

**Target Variable:**
- Wine Quality (categorical/class label)

## 🔍 Project Workflow

### 1. Data Loading & Exploration
- Loaded the wine quality dataset
- Explored dataset structure, features, and target variable
- Analyzed data distribution and correlations

### 2. Data Preprocessing
- Checked and handled missing values
- Performed feature scaling where required
- Split dataset into training and testing sets

### 3. Feature Analysis
- Studied the impact of chemical properties such as density and acidity
- Identified important predictors for wine quality

### 4. Model Building
Implemented the following classification models:
- **Random Forest Classifier**
- **Stochastic Gradient Descent (SGD) Classifier**
- **Support Vector Classifier (SVC)**

### 5. Model Evaluation
- Evaluated models using accuracy and classification metrics
- Compared performance across different classifiers

### 6. Data Visualization
- Visualized feature relationships and patterns
- Used plots to support model interpretation and insights

## 📊 Key Insights
- Certain chemical properties strongly influence wine quality
- Ensemble models like Random Forest showed strong performance
